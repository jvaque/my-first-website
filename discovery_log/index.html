Editors

Task 1: Making editors work for you

Atom shortcuts:

  • crl-shift-L = syntax highlighting menu to choose from (In case Atom doesn’t auto detect)
  • crl-, = settings menu
  • crl-shift-P = search menu
  • crl-] = indent all selected rows
  • crl-Z = undo
  • crl-Y = redo
  • crl-S = save
  • crl-F = search in the file
  • crl-/ = comment lines

Besides the shortcuts I also experimented with the fonts and themes but continued using the default one, enabled Soft Wrap since it helps to visualize the contents of the webpage. Also experimented with the snippets in order to minimize typing when making the coursework.

To help in the typing of my website I installed the atom-html-preview, but stopped using it due to it flickering to much due to the automatic update, so I used a chrome tab for the page preview. Also looked at the rest of the suggested packages and installed a few of them.
